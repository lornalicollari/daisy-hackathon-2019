# Daisy Intelligence Hackathon 2019

Algorithm that manipulates car parameters and acceleration at every second to optimize the time required for the car to traverse a variety of tracks. See problem statement [here](/res/Daisy%20Intelligence%202019%20Hackathon%20Problem%20Statement.pdf) slides [here](/out/slides.md). Placed in the top 5 teams overall. 

Built in Toronto by team Artemis, [@davidweitzenfeld](https://github.com/davidweitzenfeld) and [@lornalicollari](https://github.com/lornalicollari).
